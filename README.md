# My Kids Math Game

A small Java math game for kids, built for my university course.  
Players answer simple math questions in a fun GUI.

## Tech Stack

- Java
- Swing GUI
- Object-Oriented Programming

## Features

- Random math questions for kids
- Turn-based game flow
- Tracking correct/incorrect answers
- Simple, kid-friendly interface

## How to Run

1. Clone or download this repository.
2. Open it in an IDE (IntelliJ, NetBeans, or Eclipse).
3. Make sure all `.java` files from `My Kids Math Game` are in the same package or folder.
4. Run the `KidsGameGUI.java` (or the main class) to start the game.

## What I Learned

- Building a GUI in Java with Swing
- Organizing code using OOP (classes like `Game`, `Player`, `GameLogic`)
- Handling user input and game state
